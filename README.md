# dotnet9-skeleton
Containerized dotnet 9 WebApi using Aspire exposing minimal APIs (WeatherForecast and Customer)

Libraries used :
- Scalar
- FluentValidation
- FluentResult

Pattern used : 
- Result (with FluentResult library)
- Validation (with FluentValidation library)
- Options (with IOptionsMonitor)
